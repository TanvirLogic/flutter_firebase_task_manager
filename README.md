# Flutter Firebase Task Manager 🚀

A production-ready Flutter + Firebase app built using clean architecture principles.

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore (coming soon)
- Provider (State Management)

## Architecture

- Feature-first structure
- Domain-driven design
- Repository pattern
- Separation of concerns

## Features Implemented

- Email/Password Authentication
- Auth state listener
- Clean architecture setup
- Provider-based state management
- AuthGate navigation

## Upcoming Features

- Real-time Tasks (Firestore)
- Push Notifications
- Cloud Functions
- Testing
- CI/CD
